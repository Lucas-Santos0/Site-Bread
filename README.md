# Site sobre Pães — Versão em Inglês

Atividade sugerida em aula com o objetivo de treinar conceitos de HTML e CSS, como uso de fotos, âncoras e estilização de páginas.
Ela consistia em utilizar o mesmo site da atividade anterior (Site sobre Pão) e criar uma versão com troca de idioma (Inglês e Português).
